Brain Research Desktop App User Manual

The Ten Brain Research Desktop App offers an intuitive interface for interacting with Ten Brain Research devices. It provides real-time EEG data visualization, tracking metrics like Focus and Relaxation, and supports session recording and data export. Compatible with both Windows and Mac, the app ensures easy, wireless connectivity via a dongle.
